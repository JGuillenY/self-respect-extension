Popup Structure Changes:

  • Removed View Statistics button
  • Added three new statistics rows to the stats section:
    • Total Blocks: Cumulative count of all blocked site attempts
    • Blocks Today: Daily blocking statistics (resets at midnight)
    • Custom Domains: Number of user-added domains to block
  • Statistics update in real-time using Chrome storage listeners
  • Only one button remains: ⚙️ Open Settings for advanced configuration

Technical Implementation:

  • Storage Integration: Popup reads from chrome.storage.sync on load
  • Real-time Updates: Listens for chrome.storage.onChanged events
  • Automatic Sync: Custom domain count automatically updates in statistics
  • Backward Compatible: Maintains all existing functionality

How to Test:

  1. Load the extension in Chrome (chrome://extensions → Load unpacked → select dist folder)
  2. Click the extension icon to open the popup
  3. You should see:
    • Extension status toggle
    • Statistics section with 6 items (3 original + 3 new)
    • Protected categories list
    • "⚙️ Open Settings" button
  4. Open Settings to add custom domains
  5. Visit a blocked site (e.g., pornhub.com) to see blocking in action
  6. Return to popup to see updated statistics
