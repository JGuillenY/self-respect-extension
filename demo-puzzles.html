Self Respect Extension - Three Blocking Levels

Blocking Levels Overview

Soft Block
Gentle reminder with redirect option
  • Shows respectful overlay
  • 3-second countdown
  • Cancel option available
  • Redirect to alternative
Puzzle Block
Requires solving hard puzzles
  • Random hard puzzles
  • 5 attempts maximum
  • 10-minute time limit
  • Hint system
Hard Block
Absolute protection
  • No bypass options
  • Immediate redirect
  • Permanent blocking
  • Core values protection

Testing Instructions

  1. Build the extension: ./build.sh
  2. Load in Chrome: chrome://extensions/ → Load unpacked
  3. Open Settings → Set blocking level
  4. Visit a blocked domain (e.g., pornhub.com)
  5. Test each blocking level's behavior
